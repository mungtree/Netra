What is Mini ChatUR?

Mini ChatUR is a Rust + Tauri desktop app that orchestrates pi coding-agent processes against local code projects. In plain words: it is a control panel that runs an AI coding agent for you — you queue questions or tasks, it launches the agent, streams its output live, and saves everything. It can also run a batch: the same set of prompts across a project, then merge all the answers into one consolidated result.

It has two halves, and that split is the single most important thing to understand:

1. Headless library
A pure-Rust Cargo workspace (6 crates) with no UI dependency. Fully usable on its own — proven by the chatur command-line tool.
2. Tauri desktop shell
A thin command layer plus a SvelteKit UI that wraps the library in a desktop window.

The six crates at a glance

CrateJobDepends on
chatur-core Domain types + trait interfaces. Zero I/O. nothing internal
chatur-agent Spawns and drives pi processes over RPC. core
chatur-engine Queue, scheduler, job runner, batches, aggregation. core, agent
chatur-store SQLite persistence + file log sink. core
chatur-api The Chatur facade — the public library surface. all of the above
chatur-cli The headless chatur binary. api

Plus src-tauri (the desktop shell) and ui (the SvelteKit front-end), which both sit on top of chatur-api.
