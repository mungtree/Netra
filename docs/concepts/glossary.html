Glossary

Every term you will meet in this codebase, one line each. Keep this open in a second tab.

Domain

TermMeaning
pi The external AI coding-agent CLI (v0.74.2) that Mini ChatUR launches and drives. Not part of this repo.
Project A registered local code folder. Agents operate inside its root_path. See Project.
Job One prompt run against one project by one agent process — the smallest unit of work. See Job.
JobStatus A job's lifecycle state: Queued, Running, Completed, Failed, Cancelled.
Batch A series of prompts run across one or more targets, then merged into one result. See Batch.
BatchItem One materialized (prompt × target) pair inside a batch; each becomes a job.
BatchPrompt A named prompt in a batch — either an inline body or a reference to a template.
BatchTarget A target project for a batch, plus its {{variable}} bindings.
Map → reduce How a batch runs: map = every prompt×target becomes a job; reduce = an aggregator merges every output.
Aggregator The reduce strategy. Built-in: concat, schema_merge, reviewer.
AggregatedResult The single consolidated output of a batch's reduce step.
PromptTemplate A reusable, versioned prompt body with {{variable}} placeholders.
Prompt-set A collection of prompts that drives a batch's map step. The 8 UI task cards are prompt-sets.
ToolPolicy Which pi tools an agent may use: ReadOnly, Allowlist, or Full.
ModelRef A provider + model selection (e.g. llamacpp / qwen3.6-35b-a3b) passed to pi.
TokenUsage Input/output token counts and optional USD cost for a turn or run.

Agent & transport

TermMeaning
AgentEvent The normalized, provider-agnostic event type (TurnStart, AssistantText, ToolCall, …) that pi's raw events are mapped onto.
AgentOutput The final result of a completed job: text, optional structured JSON, token usage.
Transport The low-level link to one agent process. AgentTransport is the trait; RpcTransport drives a real pi.
Session A higher-level wrapper that runs a prompt to completion and collects an AgentOutput. Trait AgentSession; impl PiSession.
AgentSpec The launch parameters for one pi process: binary, cwd, model, tool policy, session.
AgentPool Bounds how many agents run at once, globally and per project, via semaphores.
AgentLease A checked-out agent. Releases its concurrency permits when dropped.
RPC mode pi --mode rpc — the JSON-Lines-over-stdio protocol Mini ChatUR uses.
NDJSON / JSON Lines One JSON object per line, \n-delimited. pi's wire format.
Correlation id The id on a request that pi echoes on its response, so the two can be paired.

Engine

TermMeaning
JobQueue The ordered queue of pending jobs. Trait JobQueue; impl InMemoryJobQueue.
Scheduler The loop that drains the queue into the runner under a concurrency cap.
JobRunner Executes one job: acquires an agent, streams events, retries, persists.
BatchExecutor Runs a batch: map (prompts→jobs) then reduce (aggregate).
RetryPolicy Exponential-backoff config; only Transport errors are retryable.
EventBus Broadcast pub/sub for DomainEvents. Impl BroadcastEventBus.
DomainEvent A high-level event (JobStarted, BatchCompleted, …) broadcast to observers and the UI.
OutputSink An observer of one job's event stream. Impl FileLogSink.
CancellationToken A tokio handle used to cancel a running job cooperatively.
SpecResolver Turns a job into an AgentSpec (which pi binary, model, policy). Impl ProjectSpecResolver.

Persistence & API

TermMeaning
Repository / Repo A trait that abstracts storage of one entity type (JobRepo, BatchRepo, …).
Database The SQLite connection pool; also a factory for repositories.
JSON-blob row The storage design: each entity is one JSON data column (source of truth) plus promoted columns for indexing.
Migration A versioned SQL file applied on connect to create/alter the schema.
Facade The Chatur struct — the single public surface of the headless library.
ChaturConfig TOML configuration: pi binary path, data/log dirs, concurrency, default model.
CoreError The one workspace error enum. Variants: NotFound, Invalid, Agent, Transport, Storage, Serde, Cancelled, Other.

Desktop shell

TermMeaning
Tauri The Rust desktop-app framework hosting the SvelteKit UI in a native window.
#[tauri::command] A Rust function callable from JS via invoke(). There is one per Chatur method.
chatur://event The Tauri event name that carries DomainEvents to the front-end.
Managed state The single Chatur instance Tauri holds and hands to every command.
Rune ($state) Svelte 5's reactivity primitive; the UI store is a $state object.
