chatur-core / model/aggregate.rs

One tiny type: the consolidated output of a batch's reduce step.

Path: crates/chatur-core/src/model/aggregate.rs

In plain terms

After a batch runs every prompt and collects many AgentOutputs, an aggregator merges them into a single answer. That single answer is an AggregatedResult.

Public types

⚡ Ripple effects.
  • Produced by every Aggregator::aggregate implementation and by BatchExecutor::run_reviewer.
  • Stored on Batch::result once the batch completes.
  • Rendered by the UI's LastRun component — its summary and usage fields drive that panel.
