SqliteBatchRepo — the BatchRepo trait over the batches and batch_items tables.

Path: crates/chatur-store/src/repo/batch.rs

In plain terms

The only repo that spans two tables: a batch's header in batches and its materialized items in batch_items. Both store the full struct as a JSON data blob.

⚡ Ripple effects.
  • BatchExecutor drives all seven methods: get/update for status, items at the start of the map step, update_item per mapped job.
  • The batch_items.batch_id FK is ON DELETE CASCADE; the jobs.batch_id FK is ON DELETE SET NULL — deleting a batch removes its items but leaves its jobs (just un-links them).
