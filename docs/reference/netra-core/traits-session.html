A higher-level agent interface: run a prompt to completion and get one collected output.

Path: crates/netra-core/src/traits/session.rs

In plain terms

Where AgentTransport deals in raw event streams, an AgentSession is the convenient wrapper: "here is a prompt, give me back the finished answer". It hides the stream-draining loop.
