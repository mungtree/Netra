The PromptTemplate — a reusable, versioned prompt body with placeholders.

Path: crates/netra-core/src/model/template.rs

In plain terms

Instead of retyping the same prompt, you save it once as a template with {{variable}} holes in it. A batch can reference a template by id and supply the variable values per target.
