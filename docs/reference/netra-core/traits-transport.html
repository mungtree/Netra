The low-level interface to a single agent process: send a prompt, get a stream of events.

Path: crates/netra-core/src/traits/transport.rs

In plain terms

A "transport" is the wire to one running agent. It is event-stream oriented: you hand it a prompt, it hands you back a live stream of AgentEvents. The real implementation drives pi; the test implementation replays a script.

Implementations

⚡ Ripple effects.
  • Wrapped by PiSession, which turns the event stream into a single AgentOutput.
