Crate root. Declares modules and chooses what is public.

In plain terms

A table of contents. Worth noting two deliberate choices: the protocol, pool, session, spec, transport modules are private — callers use the re-exported names, not module paths — while mock is pub so other crates can use the test doubles.

Modules

  • pub mod mock — test doubles, intentionally public.
  • mod pool / protocol / session / spec / transport — private.

Re-exports

pub use netra_core;
